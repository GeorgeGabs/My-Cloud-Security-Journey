### CIA Triad
- **C (Confidentiality):** Only right people see data
- **I (Integrity):** Data can't be secretly changed  
- **A (Availability):** System works when needed

### Security Groups
A firewall that decides who can connect and on which ports.

### SSH
A secure, encrypted way to connect to remote servers.
